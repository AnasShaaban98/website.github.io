AI Ecommerce Companion

An AI-powered tool designed to assist users in buying and selling their items on eBay.

Developed a comprehensive command-line application in Python to empower e-commerce resellers by automating the analysis of bulk listings. The tool streamlines profitability assessment by integrating with the eBay API for market data and the OpenAI API for image analysis.

Key Features:
  • AI-Powered Image Analysis: Integrated the OpenAI Vision API to interpret images of product bundles, automatically identifying and extracting a list of individual items using a multimodal Large Language Model (LLM).
  • Automated Market Valuation: Engineered an API integration module that, for each identified item, queries the eBay REST API to retrieve structured pricing data from active listings, determining an accurate estimated market value.
  • Profitability Calculation: Built a calculation engine that processes the API data to provide a detailed financial breakdown, including estimated sell price, postage, net profit, and a confidence 'Accuracy Score'.
  • Detailed Reporting: Generates a user-friendly report in a timestamped .txt file, containing a full breakdown of each item, its calculated value, and direct links to the source listings for verification.
Technologies & Skills:
  • Programming Language: Python
  • Key Skills: API Integration, Generative AI, Data Parsing
  • APIs: eBay REST API, OpenAI API (Vision)
  • Data Formats: JSON
  • Libraries: Python Requests
  • Interface: Command-Line Interface (CLI)
